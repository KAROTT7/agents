# 上线与交付检查清单

在发布 Web 应用前，至少确认以下事项。

## 产品准备

- 核心用户流程已打通
- 阻塞上线的高优缺陷已处理
- 关键指标已埋点
- 用户反馈和支持渠道已明确

## 工程准备

- 生产环境配置完成
- 环境变量与密钥已核对
- 数据备份与回滚方案已准备
- 日志、监控、告警已启用
- 基本限流和滥用防护已考虑

## 质量准备

- 冒烟测试通过
- 核心浏览器和关键设备已验证
- 基础无障碍要求已检查
- 核心流程性能在可接受范围内

## 运营准备

- 上线当天责任人明确
- 异常响应路径明确
- 发布说明或内部同步材料已准备
- 上线后观察窗口已安排
